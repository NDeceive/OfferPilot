# 智面幻境 · 智能模拟面试系统

> 基于沉浸式数字人交互与动态追问的智能模拟面试系统
> 天津工业大学 · 软件学院 · 大创项目

## 项目简介

面向大学生求职面试训练，提供「岗位选择 → 简历录入 → 数字人面试官提问 → 动态追问 → 多维评估报告 → 专项复训」的完整闭环。

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 3 + Vite + Element Plus + Pinia |
| 后端 | Java 17 + Spring Boot 3 + MyBatis-Plus |
| 数据库 | MySQL 8 |
| AI | DeepSeek（OpenAI 兼容接口）+ RAG 检索 |

## 快速启动

### 1. 数据库

确保本地 MySQL 8 已启动，然后执行建表脚本：

```bash
mysql -u root -p < 后端/src/main/resources/db/schema.sql
```

### 2. 后端

```bash
cd 后端
# 在 application.yml 中填写你的数据库密码（或设置环境变量 DB_PASSWORD）
./mvnw spring-boot:run
```

后端启动在 http://localhost:8080

### 3. 前端

```bash
cd 前端
npm install
npm run dev
```

前端启动在 http://localhost:5173

## 功能进度

- **Phase 1（已完成）**：规则化模拟面试闭环 —— 登录 → 岗位选择 → 简历画像 → 面试房间 → 提交回答 → 规则化追问 → 下一题 → 结束面试 → 历史记录。
- **Phase 2（已完成）**：规则化能力报告 —— 结束面试自动生成五维评分报告（专业知识掌握 / 项目实践表达 / 逻辑表达能力 / 岗位匹配程度 / 动态追问应对），Report.vue 提供总分卡 + ECharts 雷达图 + 优势 / 不足 / 建议 + 维度解释，历史记录可一键查看报告。暂未接入 DeepSeek，评分为确定性规则引擎。

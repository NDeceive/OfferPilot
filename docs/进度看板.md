## Phase 进展记录

### Phase 1 · 规则化面试闭环（已完成）
- 题库种子（Java 后端 15 题 / Web 前端 10 题）
- 开始 → 作答 → 规则化追问 → 下一题 → 结束 全流程跑通
- 面试记录页（History.vue）展示真实历史会话

### Phase 2 · 规则化能力报告（已完成）
- 五维评分模型：专业知识掌握 30% / 项目实践表达 25% / 逻辑表达能力 20% / 岗位匹配程度 15% / 动态追问应对 10%
- 结束面试自动生成 1 份报告 + 5 条维度明细（确定性规则打分，幂等）
- `GET /api/report/{reportId}` 报告详情接口（含归属校验）
- Report.vue 真实报告页：总分卡 + ECharts 五维雷达图 + 优势/不足/建议 + 维度解释卡
- History.vue「查看报告」按钮在报告生成后可点击跳转
- 暂不接入 DeepSeek，规则引擎为后续 AI 评分预留接口位
